
# Fiducial test task




## FAQ

#### For running locally:

Install .net SDK v7.0 from https://dotnet.microsoft.com/en-us/download/dotnet/7.0



## Running the application
Using bash or powershell

```bash
  cd fiducial.web
  dotnet restore
  dotnet run
```
    
## Running Tests

To run tests, run the following command

```bash
  cd fiducial.tests
  dotnet test
```
